# Formats

### Minimal
```
Table Heading
col1      │ col2      │ col3      │ col4
──────────┼───────────┼───────────┼──────────
entry 1.1 │ entry 1.2 │ entry 1.3 │ entry 1.4
entry 2.1 │ entry 2.2 │ entry 2.3 │ entry 2.4
entry 3.1 │ entry 3.2 │ entry 3.3 │ entry 3.4
```
### Default:
```
┌────────────────────────────────────────────────┐
│ Table Heading                                  │
├────────────┬───────────┬───────────┬───────────┤
│ col1       │ col2      │ col3      │ col4      │
├────────────┼───────────┼───────────┼───────────┤
│ entry 1.1  │ entry 1.2 │ entry 1.3 │ entry 1.4 │
│ entry 2.1  │ entry 2.2 │ entry 2.3 │ entry 2.4 │
│ entry 3.1  │ entry 3.2 │ entry 3.3 │ entry 3.4 │
└────────────┴───────────┴───────────┴───────────┘
```
